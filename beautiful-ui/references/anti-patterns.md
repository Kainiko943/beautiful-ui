# Anti-Patterns

- Generic SaaS dashboards with no product point of view.
- Random gradients and shadows used instead of hierarchy.
- Default fonts with no typography personality.
- Card piles where every section floats.
- Designing only the happy path.
- Skipping focus, disabled, validation, empty, loading, and error states.
- Forcing the same chrome on every platform.
- Inventing components before checking the repo.
- Treating inspiration images or Figma files as unquestionable truth.
