# Platform Selection

- Use `platform-web-ui` for responsive browser products, dashboards, sites, and web apps.
- Use `platform-ios-ui` for iPhone or iPad-first native apps.
- Use `platform-android-ui` for Android-first native apps.
- Use multiple adapters only when the deliverable must coordinate platforms.
- If the repo reveals the target platform, do not ask the user to restate it.
- Ask one short clarifying question only when platform ambiguity changes the design.
