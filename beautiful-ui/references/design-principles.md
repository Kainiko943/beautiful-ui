# Design Principles

## Hierarchy

Make the primary job obvious. Use type, spacing, alignment, contrast, and content order before decoration.

## Typography

Choose a type personality that fits the product. Use a visible scale and avoid oversized type inside dense tools.

## Color

Use color to signal meaning, priority, and brand. Avoid one-note palettes and arbitrary gradients.

## Spacing

Use a repeatable rhythm. Increase spacing to clarify groups, not to make screens feel empty.

## Surfaces

Let surfaces explain structure. Cards are for repeated items, modals, and framed tools, not every section.

## Motion

Use motion to explain continuity, feedback, and state change. Keep it brief and respect reduced motion.
