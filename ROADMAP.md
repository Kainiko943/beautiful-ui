# Roadmap

## Month 1: Credible artifact

- Ship the core `beautiful-ui` skill.
- Ship web, iOS, Android, and verification skills.
- Publish five before/after examples.
- Document install paths for major agent surfaces.
- Add contribution and issue templates.

## Months 2-3: Launch loop

- Launch on developer and AI communities.
- Publish short posts about generic AI UI and the skill workflow.
- Recruit contributors for examples, adapters, docs, and verification.
- Track stars, clones, PRs, and example reuse.

## Months 4-6: Make it hard to ignore

- Add framework guides for shadcn, Tailwind, SwiftUI, UIKit, Android Compose, Flutter, and React Native.
- Add a gallery site generated from examples.
- Add agent evals showing output with and without the skill.

## Months 7-9: Ecosystem expansion

- Add desktop and cross-platform adapters after core/mobile/web prove useful.
- Add community showcases.
- Publish monthly UI rescue examples.

## Months 10-12: Scale the flywheel

- Stabilize v1.0 package structure.
- Add contributor credits and usage badge.
- Relaunch with measured improvements from examples and evals.
