# Security Policy

Beautiful UI is a prompt and skill package, not an application runtime. Security issues may still appear in install instructions, examples, or generated guidance.

## Reporting

Please open a GitHub issue for documentation or prompt-safety concerns that are not sensitive.

For sensitive reports, contact the maintainer privately through the GitHub profile associated with this repository.

## Scope

- Skill instructions that encourage unsafe implementation patterns.
- Install guidance that risks leaking credentials.
- Examples that recommend insecure UI or authentication behavior.
