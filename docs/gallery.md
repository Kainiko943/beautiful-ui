# Example Gallery

The gallery is intentionally prompt-led in v0.1. Each example shows how a weak prompt becomes a stronger design handoff.

| Example | Weak prompt problem | Skill outcome |
| --- | --- | --- |
| SaaS dashboard | Generic cards and charts | Revenue-focused hierarchy, filters, responsive states |
| Mobile finance app | Flashy fintech surface | Trust-focused native mobile flow |
| Admin table | Decorative redesign | Dense, keyboard-friendly review queue |
| Landing page | Vague hero and card pile | Product-led conversion hierarchy |
| Cinematic web experience | Cool website with no art direction | WebGL/Three.js-ready cinematic hero, motion model, fallbacks |
| Settings forms flow | Flat forms | Clear save, validation, and destructive-action states |

Future gallery versions should add rendered screenshots and agent evals comparing outputs with and without the skill.
